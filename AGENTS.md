# AGENTS.md

This repository keeps its agent operating contract in **[`CLAUDE.md`](CLAUDE.md)**.

Whatever tool you are — Claude Code, Cursor, Aider, Copilot, or any other agent that
looks for an `AGENTS.md` — read `CLAUDE.md` first and follow it. It is the single source
of truth for *how* work is done here.

The two rules that override everything else (full detail in `CLAUDE.md` §0):

1. **This is a learning project.** After completing any task, your reply must include a
   **"How & Why"** section explaining what you did, how it works, why you chose that
   approach, and what it costs.
2. **Code documentation is concise and explains *why*, not *what*** — so the owner can
   reopen any file later and learn the reasoning from the comments alone.

For product scope, read [`README.md`](README.md). For architecture, read
[`docs/ARCHITECTURE.md`](docs/ARCHITECTURE.md). For the ordered work, read
[`BACKLOG.md`](BACKLOG.md). For traps already discovered, read [`LESSONS.md`](LESSONS.md).
